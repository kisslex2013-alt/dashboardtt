Рекомендуемое решение: PocketBase
PocketBase — это самый простой вариант для zero-coder с автоматической авторизацией и синхронизацией. Он предоставляет бесплатный self-hosted backend с встроенной аутентификацией и файловым хранилищем.
​

Реализация с PocketBase
Подключение и авторизация:

javascript
import PocketBase from 'pocketbase';

const pb = new PocketBase('https://your-app.pockethost.io');

// Авторизация через email (1 клик)
async function login(email, password) {
  const authData = await pb.collection('users').authWithPassword(email, password);
  // Токен автоматически сохраняется в localStorage
  return authData;
}

// Регистрация нового пользователя
async function signup(email, password) {
  await pb.collection('users').create({
    email: email,
    password: password,
    passwordConfirm: password
  });
  return login(email, password);
}
Сохранение и загрузка JSON данных:

javascript
// Сохранение записей времени
async function saveTimeRecords(data) {
  const jsonBlob = new Blob([JSON.stringify(data)], { type: 'application/json' });
  const formData = new FormData();
  formData.append('data', jsonBlob, 'time_records.json');
  
  return await pb.collection('user_data').create({
    userId: pb.authStore.model.id,
    dataFile: formData.get('data')
  });
}

// Загрузка данных при входе
async function loadTimeRecords() {
  const records = await pb.collection('user_data').getList(1, 1, {
    filter: `userId="${pb.authStore.model.id}"`,
    sort: '-created'
  });
  
  if (records.items.length > 0) {
    const fileUrl = pb.files.getUrl(records.items[0], records.items[0].dataFile);
    const response = await fetch(fileUrl);
    return await response.json();
  }
  return null;
}
Автоматическая синхронизация:

// Real-time синхронизация между устройствами

Альтернатива 1: Supabase
Supabase предоставляет бесплатный tier (500 MB хранилища) с простым JavaScript SDK и встроенной OAuth авторизацией.
​

Настройка:

javascript
import { createClient } from '@supabase/supabase-js';

const supabase = createClient(
  'https://your-project.supabase.co',
  'your-anon-key'
);

// Авторизация через email
async function signIn(email, password) {
  const { data, error } = await supabase.auth.signInWithPassword({
    email: email,
    password: password
  });
  return data;
}

// Сохранение JSON файла
async function saveData(jsonData) {
  const blob = new Blob([JSON.stringify(jsonData)], { type: 'application/json' });
  const fileName = `${supabase.auth.user().id}/data.json`;
  
  const { data, error } = await supabase.storage
    .from('user-data')
    .upload(fileName, blob, { upsert: true });
  
  return data;
}

// Загрузка данных
async function loadData() {
  const fileName = `${supabase.auth.user().id}/data.json`;
  const { data, error } = await supabase.storage
    .from('user-data')
    .download(fileName);
  
  const text = await data.text();
  return JSON.parse(text);
}
Supabase автоматически изолирует данные по пользователям через Row Level Security.

  return await account.createEmailSession(email, password);
}

// Сохранение JSON
async function saveTimeData(jsonData) {
  const blob = new Blob([JSON.stringify(jsonData)], { type: 'application/json' });
  const file = new File([blob], 'timedata.json');
  
  return await storage.createFile('user-bucket', ID.unique(), file);
}

// Загрузка данных
async function loadTimeData() {
  const files = await storage.listFiles('user-bucket');
  if (files.files.length > 0) {
    const fileId = files.files[0].$id;
    const result = await storage.getFileDownload('user-bucket', fileId);
    return JSON.parse(await result.text());
  }
}
Appwrite автоматически привязывает файлы к авторизованному пользователю.

// Сохранение JSON на Drive
async function saveToDrive(jsonData) {
  const fileMetadata = {
    name: 'timetracking.json',
    mimeType: 'application/json'
  };
  
  const form = new FormData();
  form.append('metadata', new Blob([JSON.stringify(fileMetadata)], { type: 'application/json' }));
  form.append('file', new Blob([JSON.stringify(jsonData)], { type: 'application/json' }));
  
  const response = await fetch(
    'https://www.googleapis.com/upload/drive/v3/files?uploadType=multipart',
    {
      method: 'POST',
      headers: new Headers({ 'Authorization': 'Bearer ' + gapi.client.getToken().access_token }),
      body: form
    }
  );
  
  return await response.json();
}

как PocketBase будет выглядеть для конечного пользователя?
5 шагов выполнено
PocketBase — это backend-решение, которое само по себе не предоставляет готовый интерфейс для конечных пользователей. Вам нужно создать UI самостоятельно, но для zero-coder это можно сделать максимально просто.

Что видит пользователь пошагово
1. Первый запуск приложения:
Пользователь видит простую форму с двумя полями (Email и Пароль) и двумя кнопками: "Войти" и "Зарегистрироваться".
​

2. Регистрация (1 клик):
Пользователь вводит email и пароль, нажимает "Зарегистрироваться" — аккаунт создаётся автоматически.
​

3. Вход (1 клик):
При следующем посещении пользователь просто вводит данные и нажимает "Войти".
​
​

4. Автоматическая загрузка данных:
После входа приложение автоматически загружает персональные данные пользователя (записи времени, настройки).
​

5. Синхронизация между устройствами:
Если пользователь откроет приложение на другом устройстве и войдёт под тем же email — все данные синхронизируются автоматически.

Важные отличия от других решений
Что видит пользователь	PocketBase	Firebase	Supabase
Форма входа	Ваш дизайн	Готовая форма или свой дизайн	Свой дизайн
Регистрация	Email + пароль	Email + пароль или OAuth	Email + пароль
Брендирование	100% ваше	Может быть Firebase UI	100% ваше
Сложность реализации	50 строк HTML	SDK + конфигурация	SDK + конфигурация
Admin UI vs User UI
Важно понимать: PocketBase имеет встроенный Admin UI (доступен по адресу http://127.0.0.1:8090/_/), но это только для вас как администратора. Конечные пользователи никогда не видят этот интерфейс.
​

Admin UI используется для:

Создания таблиц данных

Настройки прав доступа

Просмотра зарегистрированных пользователей

Конфигурации OAuth провайдеров
​

User UI (ваше приложение):

Полностью ваш дизайн

Простые HTML формы

Интеграция с вашим приложением учёта времени
