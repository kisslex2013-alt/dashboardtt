# Руководство: Настройка своего Backend (PocketBase)

Вы можете выбрать один из двух вариантов: **Облако** (удобно, но может быть платно) или **Локальный сервер** (бесплатно, но работает только на вашем ПК).

---

## Вариант 1: Облачный хостинг (PocketHost.io)
*Подходит, если нужен доступ с телефона и других устройств.*

> **Внимание:** У сервиса PocketHost бывают изменения в тарифах. Если он требует оплату, используйте Вариант 2.

1.  Перейдите на [https://pockethost.io/](https://pockethost.io/) и создайте аккаунт.
2.  Создайте новый Backend (придумайте имя, выберите локацию).
3.  URL вашего сервера будет: `https://ваше-имя.pockethost.io`

---

## Вариант 2: Локальный сервер (Бесплатно навсегда)
*Идеально для хранения бэкапов. Бесплатно. Данные хранятся только у вас.*

### 1. Скачайте PocketBase
1.  Скачайте файл для Windows: [https://github.com/pocketbase/pocketbase/releases/download/v0.24.4/pocketbase_0.24.4_windows_amd64.zip](https://github.com/pocketbase/pocketbase/releases/download/v0.24.4/pocketbase_0.24.4_windows_amd64.zip)
2.  Распакуйте архив в любую папку (например, создайте папку `backup_server` на Рабочем столе).

### 2. Запустите сервер
1.  Откройте эту папку.
2.  Зажмите `Shift` и щелкните правой кнопкой мыши по пустому месту в папке -> **"Открыть окно PowerShell здесь"** (или терминал).
3.  Введите команду:
    ```powershell
    .\pocketbase.exe serve
    ```
4.  Не закрывайте это окно! Сервер работает, пока окно открыто.
    Ваш локальный адрес: `http://127.0.0.1:8090`

---

## Вариант 3: Настройка структуры данных (Для любого варианта)

1.  Откройте админку в браузере:
    *   Если Облако: `https://ваше-имя.pockethost.io/_/`
    *   Если Локально: `http://127.0.0.1:8090/_/`
2.  Создайте первого админа (email/пароль).
3.  В меню слева перейдите в **Settings** (шестеренка) -> **Import collections**.
4.  Нажмите **"Load from JSON file"** и выберите файл `docs/pb_schema.json` из папки этого проекта.
5.  Нажмите **"Import"**.

---

## Шаг 4. Подключение к приложению

1.  Откройте файл `src/services/pocketbase.ts`.
2.  Отредактируйте переменную `POCKETBASE_URL` (строка 5):

**Для локального сервера:**
```typescript
const POCKETBASE_URL = 'http://127.0.0.1:8090'
```

**Для облака:**
```typescript
const POCKETBASE_URL = 'https://ваше-имя.pockethost.io'
```

3.  Сохраните файл. Перезагрузите страницу приложения.
