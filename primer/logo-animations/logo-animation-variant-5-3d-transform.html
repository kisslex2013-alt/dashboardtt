Анимация логотипа - Вариант 5

3D пульсация

Каждый элемент анимируется независимо с легким 3D эффектом

Логотип с анимацией

Описание анимации:

  • • Пульс (волна данных) - динамичная бегущая волна (эффект движения слева направо)
  • • 4 концентрических круга - пульсируют независимо с 3D эффектом (rotateX)
  • • Стрелка часов - пульсирует с легким вращением (rotateZ)
  • • Точки на пульсе - пульсируют (scale 1.0 → 1.6)
  • • Каждый элемент анимируется отдельно

CSS код:

/* Пульс с 3D эффектом */
@keyframes pulse-heartbeat {
  0%, 100% { transform: scaleX(1) rotateY(0deg); }
  50% { transform: scaleX(1.35) rotateY(0deg); }
}

/* Круги с 3D эффектом */
.logo-animation-5 svg .circle-1 {
  animation: circle-pulse-1 2.2s ease-in-out infinite;
}

.logo-animation-5 svg {
  transform-style: preserve-3d;
}
