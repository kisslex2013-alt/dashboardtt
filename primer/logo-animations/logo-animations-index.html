Как использовать

Каждый вариант анимации можно использовать в проекте, скопировав CSS код из соответствующей страницы.

Все элементы логотипа анимируются независимо:

  • Пульс (волна данных) - пульсирует как сердце (scaleX)
  • Круги (4 концентрических) - пульсируют независимо (scale)
  • Стрелка (часы) - пульсирует отдельно (scale)
  • Точки (на пульсе) - пульсируют отдельно (scale)

Все анимации оптимизированы для производительности.
