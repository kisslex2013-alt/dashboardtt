Анимация логотипа - Вариант 3

Градиентная пульсация

Каждый элемент анимируется независимо с градиентным изменением цвета

Логотип с анимацией

Описание анимации:

  • • Пульс (волна данных) - резкие острые пики (как на ECG с аритмией)
  • • 4 концентрических круга - пульсируют независимо с изменением цвета
  • • Стрелка часов - пульсирует (scale 1.0 → 1.22)
  • • Точки на пульсе - пульсируют (scale 1.0 → 1.55)
  • • Каждый элемент анимируется отдельно

CSS код:

/* Пульс с градиентным изменением цвета */
@keyframes pulse-heartbeat {
  0%, 100% { transform: scaleX(1); filter: hue-rotate(0deg); }
  50% { transform: scaleX(1.4); filter: hue-rotate(180deg); }
}

/* Круги - пульсируют независимо */
.logo-animation-3 svg .circle-1 {
  animation: circle-pulse-1 2.5s ease-in-out infinite;
}
