Анимация логотипа - Вариант 4

Интенсивное свечение

Каждый элемент анимируется независимо с интенсивным оранжевым свечением

Логотип с анимацией

Описание анимации:

  • • Пульс (волна данных) - сложная волна с несколькими пиками (комплекс QRS)
  • • 4 концентрических круга - пульсируют независимо с ярким свечением
  • • Стрелка часов - пульсирует с эффектом свечения (scale 1.0 → 1.3)
  • • Точки на пульсе - пульсируют с ярким свечением (scale 1.0 → 1.7)
  • • Каждый элемент анимируется отдельно

CSS код:

/* Пульс с интенсивным свечением */
@keyframes pulse-heartbeat {
  0%, 100% { 
    transform: scaleX(1); 
    filter: drop-shadow(0 0 10px rgba(245, 158, 11, 0.5));
  }
  50% { 
    transform: scaleX(1.4); 
    filter: drop-shadow(0 0 25px rgba(245, 158, 11, 0.9));
  }
}

.logo-animation-4 svg .pulse-path {
  animation: pulse-heartbeat 1.2s ease-in-out infinite;
}
