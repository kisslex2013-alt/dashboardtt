Оптимизация для фавикона:

  • • Упрощенная геометрия - меньше деталей, больше контраста
  • • Толстые линии (stroke-width: 12, 10, 8) - видны на малых размерах
  • • Яркие цвета - градиент синий→зеленый, оранжевый акцент
  • • Заметная пульсация - scale 1.0 → 1.15
  • • Мощное свечение - drop-shadow с большим радиусом
  • • Быстрое вращение - 2 секунды на оборот
  • • Мигающий центр - привлекает внимание

Как использовать:

<!-- В HTML -->
<link rel="icon" type="image/svg+xml" href="/favicon.svg">

<!-- Создайте favicon.svg с этим кодом -->
<svg viewBox="0 0 200 200" xmlns="http://www.w3.org/2000/svg">
  <style>
    @keyframes pulse { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.15); } }
    @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }
    @keyframes blink { 0%, 100% { opacity: 1; } 50% { opacity: 0.3; } }
    .main-circle { animation: pulse 1.5s ease-in-out infinite; transform-origin: 100px 100px; }
    .spinner { animation: spin 2s linear infinite; transform-origin: 100px 100px; }
    .center-dot { animation: blink 1s ease-in-out infinite; }
  </style>
  <!-- SVG content here -->
</svg>
