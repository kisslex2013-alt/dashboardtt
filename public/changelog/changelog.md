## [1.3.3] - 2026-01-05

#### Новые возможности

## [1.3.2] - 2026-01-02

### 🚀 **Глобальное обновление аналитики и интерфейса**

#### Новые возможности

- 📊 Продвинутая система аналитики — полный спектр инструментов для понимания вашего роста:
  - **Базовая**: Ключевые показатели эффективности в реальном времени.
  - **Предиктивная**: Умные прогнозы будущего дохода и предупреждения о риске выгорания.
  - **Описательная**: Детальный анализ структуры времени и задач.
  - **Сравнительная**: Наглядное сопоставление периодов для оценки прогресса.
- 📅 **Календарь доходов** — новый визуальный формат работы с данными. Оценивайте успешность дней с одного взгляда благодаря цветовой индикации.
- 🤖 **Умные AI-уведомления** — теперь в отдельном разделе. Гибкая настройка позволит получать только самые важные инсайты и советы, не отвлекаясь от работы.
- 🛡️ **Система безопасности данных** — новая функция логирования сохраняет историю всех изменений, обеспечивая надежность и возможность восстановления.

## [1.3.0] - 2025-12-29

### ⚡ **ОБНОВЛЕНИЕ: Архитектура AI-анализа (Performance)**

#### Новые возможности

- 🧠 **Web Worker для AI-анализа** — тяжелые вычисления (риск выгорания, прогнозы, аномалии) перенесены в фоновый поток. Интерфейс больше не блокируется при анализе больших объемов данных.
- 🔄 **Умное кэширование** — анализ запускается только при изменении записей или целей (сигнатура: `entries.length + lastEntry.date + dailyGoal`).
- ⏱️ **Индикатор "Последний анализ"** — в настройках AI-уведомлений теперь отображается время последнего анализа (например, "2 минуты назад").

#### Технические улучшения

- 🔧 **Новый файл**: `src/workers/aiAnalysisWorker.ts` — Worker для AI-анализа.

- 📈 **Оптимизация** `StatisticsDashboard.tsx` — вынесена логика расчетов в `statisticsCalculations.ts`.
- 📅 **Оптимизация** `CalendarHeatmap.tsx` — создан мемоизированный `CalendarMonth` компонент.

#### Техническая информация

- **Версия**: 1.3.0
- **Файлы**: `src/workers/aiAnalysisWorker.ts`, `src/hooks/useAINotificationMonitor.ts`, `src/store/useAINotificationsStore.ts`, `src/components/modals/AINotificationsSettingsModal.tsx`, `src/components/charts/CalendarMonth.tsx`
