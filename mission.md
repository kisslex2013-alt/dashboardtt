# 🎯 Миссия проекта: Time Tracker Dashboard

## Описание
**Time Tracker** — умный учёт рабочего времени с аналитикой. Веб-приложение для отслеживания времени работы, категоризации задач, анализа продуктивности и расчёта заработка.

## Технологии
- **Frontend**: React 18 + TypeScript
- **State Management**: Zustand
- **Styling**: Tailwind CSS + Custom CSS
- **Animations**: Framer Motion
- **Build**: Vite
- **Testing**: Vitest + Playwright

## Основные модули
- **Entries** — записи времени (добавление, редактирование, удаление)
- **Statistics** — статистика и графики (Recharts)
- **Pomodoro** — таймер помидора
- **Settings** — настройки (категории, цвета, расписание)
- **Modals** — модальные окна (импорт/экспорт, настройки)

## Критерии качества
1. ✅ Надёжность — приложение работает без сбоев
2. ✅ Плавность — анимации 60fps, без лагов
3. ✅ Отзывчивость — мгновенная реакция на действия пользователя
4. ✅ Оффлайн — работает без интернета (PWA)
